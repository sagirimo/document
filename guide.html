      # 文档站 MagicDocs 使用导论

      # 一、MagicDocs 有什么功能 ？

      知识分享对于学生来说很重要，写博客有很多好处：

      • 帮助自己梳理、总结专业课程的知识点，大二以后会逐渐上传临床专业知识（个人提升）
      • 在实践中学习前端开发，学习JavaScript等语言（自学素养）
      • 作为 pcgame club 的文章分享根据地（摸鱼快乐）
      • 帮助别人理解知识点，获取更多信息（用爱发电）

      # 二、怎么正确使用 MagicDocs ？

      MagicDocs 有三大板块，卷无止境、躺平时光、芝士雪豹。接下来我会逐一解释三大板块的内容

      # 卷无止境

      此板块是专业知识板块，主要面向PKU临床五年制同学设置。 目前包含的学科有：

      • 军事理论
      • 有机化学
      • 大学物理

      # 躺平时光

      此板块服务于日照一中PC游戏社，以及其他爱好游戏的同志。 板块内容可投稿。 目前包含的内容有：

      • 游戏测评
      • 外设测评
      • 游戏感悟
      • 等等

      # 芝士雪豹

      此板块是课外知识板块，包括除专业课以外的知识。 内容主要是站主的学习笔记，为分享知识而设立
